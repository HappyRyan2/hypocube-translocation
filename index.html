Hypocube Translocation


This is an abstract puzzle game about translocating (moving) hypocubes (squares). In each level, your task is to translocate the blue square onto the red square. The game is played in a grid of tiles containing extenders that, when clicked, extend to push (or sometimes pull) objects in a certain direction.
